# 03_Multiplayer_Fortress – Tile Palette

This palette defines the structural tiles and material variations for the Fortress Deux style map, intended as a high-fidelity evolution of the N64 War/Total War arena.

## Floor Tiles

T_Floor_IndustrialPlain
- Use: Base ground in courtyards and tower interiors.
- Variant A: Dry concrete with faint tire tracks.
- Variant B: Mud-streaked version for trenches and low ground.

T_Floor_IndustrialHazardStripe
- Use: Bridge edges, tower entry thresholds, and capture zones.
- Variant A: Yellow/black clean stripes.
- Variant B: Faded stripes with chipped paint.

T_Floor_MetalGrate
- Use: Catwalks between towers, overhangs above the central valley.
- Variant A: Standard steel grate.
- Variant B: Reinforced grate with cross-bracing.

T_Floor_TrenchMud
- Use: Inside trenches and shell craters.
- Variant A: Wet mud with puddles.
- Variant B: Scorched mud with embedded shell casings.

## Wall / Tower Tiles

T_Wall_ConcreteBunker
- Use: Exterior of bases and trench walls.
- Variant A: Plain concrete.
- Variant B: Concrete with bullet impact decals baked into texture.

T_Wall_TowerPanel
- Use: Tower exterior skins.
- Variant A: Clean riveted plates.
- Variant B: Plates with rust streaks below bolt lines.

T_Wall_TowerWindow
- Use: Firing slits and sniper nest windows.
- Variant A: Narrow horizontal slit.
- Variant B: Full window with metal bars.

## Ceiling / Overhead

T_Ceiling_TunnelRibs
- Use: Interior of tunnels connecting bases.
- Variant A: Ribbed metal with conduit runs.
- Variant B: Cracked concrete with exposed rebar.

T_Ceiling_GratedCatwalk
- Use: Under-side of suspended walkways.
- Single variant; reuse normal floor grate normals.

## Special Objective Tiles

T_Floor_CaptureZone
- Use: Ground under capture flags and key tower control points.
- Variant A: Painted circle with faction emblem.
- Variant B: Neutral white ring for contested towers.

T_Floor_SpawnZone
- Use: Spawn room floor tiles.
- Single variant with subtle iconography but same palette as base floor.

All tiles share the same world scale (4x4 units) and color palette as Alien Base and Beach to preserve the “shared universe” look while clearly communicating Fortress Deux’s heavier war-metal identity.
